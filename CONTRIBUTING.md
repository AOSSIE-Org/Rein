# Contributing to Rein

Thank you for your interest in contributing to Rein.

## Communication

Project communication happens on the AOSSIE Discord server.
Please share your pull request link in the project channel after opening a PR.

## Getting started

### Prerequisites

- Node.js (LTS)
- npm

### Fork and clone

git clone https://github.com/<your-username>/Rein.git
cd Rein

Add upstream:

git remote add upstream https://github.com/AOSSIE-Org/Rein.git

## Making changes

Create a new branch:

git checkout -b your-branch-name

Rein currently does not have automated tests.
Please test your changes manually by running the app.

## Submitting a pull request

1. Push your branch to your fork
2. Open a PR to AOSSIE-Org/Rein
3. Link the related issue
4. Share the PR in Discord
